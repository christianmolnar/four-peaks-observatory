# Setup & Configuration Documentation

This directory contains setup and configuration guides for various aspects of the Maple Valley Observatory website.

## Files

### `CONTACT_SETUP.md`
Contact form configuration and setup:
- Contact form implementation
- Email integration setup
- Contact information management
- Social media links configuration

### `SEO_SETUP_GUIDE.md`
Search engine optimization configuration:
- Meta tags setup
- SEO best practices
- Search engine indexing
- Performance optimization for SEO

### `agent-setup-instructions.md`
AI agent and development environment setup:
- Beast Mode configuration
- shadcn/ui setup instructions
- Agent memory configuration
- Development tools setup

## Purpose

These guides provide step-by-step instructions for:
- Initial website setup and configuration
- Third-party service integrations
- Development environment preparation
- Optimization and performance tuning

## Usage

Use these guides when:
- Setting up a new development environment
- Configuring contact forms and SEO
- Integrating new services or tools
- Troubleshooting setup issues

## Gallery Ordering

All galleries automatically sort images **newest first** based on the `dateTaken` field in metadata.json:
- Newest images appear at the top of each gallery page
- Sorting uses metadata dates (e.g., "August, 2025") rather than file timestamps
- Ensures latest uploads and observations are prominently displayed
- Fallback to file modification times when metadata dates are unavailable
